# 🏡 Wanderlust
**A full-stack Airbnb-inspired property booking platform built using the MERN backend ecosystem with secure authentication, image uploads, interactive maps, reviews, and role-based authorization.**

## Tech Stack

| Category       | Technology               |
| -------------- | ------------------------ |
| Backend        | Node.js                  |
| Framework      | Express.js               |
| Database       | MongoDB Atlas            |
| ODM            | Mongoose                 |
| Authentication | Passport.js              |
| Authorization  | Passport Local           |
| Image Storage  | Cloudinary               |
| Uploads        | Multer                   |
| Maps           | MapLibre + OpenStreetMap |
| Session Store  | connect-mongo            |
| Templating     | EJS                      |
| Styling        | Bootstrap 5              |
| Deployment     | Render                   |

<br>

## Project Highlights

• Built using MVC Architecture

• Secure Authentication using Passport.js

• Authorization for Listings and Reviews

• Cloudinary Image Upload

• Interactive Maps using MapLibre

• Session-based Authentication

• Flash Messaging

• Server-side Validation

• Centralized Error Handling

• Responsive Bootstrap UI

• MongoDB Relationships

• RESTful Routing
